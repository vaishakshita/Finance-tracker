## Finance Tracker

A full-stack finance management application built with Next.js, React, Node.js, Express, and MongoDB


## Features Implemented

- User Authentication (Login/Signup)
- Dashboard
- Transactions Page
- Income & Expense Tracking

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
